# The Solar System

Fetching from the database with Graphql, resolve the data 
with vanila Javascript and combine the DOM with the CSS 
for the final appearance of this web site! 

------------------------
 Web Technologies

Client Side
- Html
- Css
- Javascript ( Javascript Dom )
     
Server Side
- Express
- GraphQl
- Apollo Server Express
- Sequelize

Database
- Mysql

-----------------------------
